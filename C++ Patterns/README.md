# C++ Pattern Programs 🔷

A collection of C++ pattern programs using loops — great for beginners learning nested loops and logic building.

## Patterns List

| File | Pattern Name | Description |
|------|-------------|-------------|
| `01_alphabet_triangle.cpp` | Alphabet Triangle | Prints A, AB, ABC... on each row |
| `02_continuous_alphabet_grid.cpp` | Continuous Alphabet Grid | Prints alphabets continuously across rows |
| `03_left_aligned_star_triangle.cpp` | Left Aligned Star Triangle | Grows from 1 star to n stars |
| `04_inverted_star_triangle.cpp` | Inverted Star Triangle | Shrinks from n stars to 1 star |
| `05_hourglass_star.cpp` | Hourglass Star | Inverted triangle on top + normal triangle below |
| `06_right_aligned_growing_triangle.cpp` | Right Aligned Growing Triangle | Stars grow and align to the right |
| `07_right_aligned_shrinking_triangle.cpp` | Right Aligned Shrinking Triangle | Stars shrink with right alignment |
| `08_right_aligned_spaced_star_triangle.cpp` | Right Aligned Spaced Star Triangle | Stars with spaces, right aligned |

## How to Compile & Run

```bash
g++ filename.cpp -o output
./output
```

Example:
```bash
g++ 01_alphabet_triangle.cpp -o pattern1
./pattern1
```
